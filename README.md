# custom-two-row-slider
A simple image gallery slider to display images in two rows
## Usage:
 Example: [two_row_slider images="1,2,3,4,5,6,7,8" title="Your Slider Title"]
### Notes: 
1) It uses the WordPress Image Media IDs in a list;
2) Feel free to fork it and PR;
3) This was a quick solution using AI to display logos in multiple rows.
